# 🌦️ Weather App

A simple and responsive Weather App built with HTML, CSS, and JavaScript using the OpenWeather API.  
It allows users to search for real-time weather information by city name.

---

## 🚀 Features

- 🔍 Search weather by city
- 🌡️ Real-time temperature (°C)
- ☁️ Weather description (clouds, rain, etc.)
- 💧 Humidity and wind speed
- ⏳ Loading state
- ❌ Error handling (invalid city / network error)
- 🎨 Clean and responsive UI

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- OpenWeather API
